I built Minimalipsum as a side project to practice Node.js. It is a minimal interface for copying Lorem Ipsum text to your clipboard.

I always enjoy learning new technologies. This was my first project using Node.js, so I found it particularly interesting.

There isn't much to the app, but that's what I was going for. I wanted it to be as minimally intrusive as possible.
