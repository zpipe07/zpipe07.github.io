I developed the front end for SolarCity's operation careers page. It is built on top of Drupal.

This project was fun to develop. I enjoyed working with such a unique and modern design, high quality assets, and building a highly interactive interface.

I was lucky enough to work a lot with Vimeo's API. With just a video's URL I was able to easily pull in videos' thumbnails, titles, and iframes with AJAX.

The site has some interactive UI features including the floating sidebar navigation and the sortable and searchable job listing page, both of which are accomplished with JavaScript.
